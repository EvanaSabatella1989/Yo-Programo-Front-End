Aplicación web full stack, que muestra misdatos personales, estudios cursados, experiencia laboral, conocimiento de las tecnologías y demas. Es una aplicación de arquitectura distribuida y conteniene un diseño de interfaz de usuario (front end) que muestra la información, una base de datos que almacena los datos antes mencionados y cuenta con las APIs necesarias para proveer a través de internet la información (back end).

Pagina Web de un Gym desarrollado en el 2021. El proyecto contiene etiquetas semanticas para estructurar el sitio de forma correcta, un formulario de contacto con validacion mediante JavaScript. Ademas se utilizaron iframes, iconos de FontAwesome y fuentes tanto locales como de Google Fonts. El proyecto es totalmente responsivo, con 3 puntos de corte para 3 tamaños de dispositivos(PC escritorio, Tablet, Celular)
